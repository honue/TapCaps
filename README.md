默认功能：实现Windows下 使用 Caps 大小写按键切换逻辑，参照 Macos 切换输入法的逻辑

主要用于重定义 CapsLock 键的功能。它通过检测按键时长（短按 vs 长按）和当前的大写状态，来实现输入法切换与大写锁定的智能控制。

### 1. 核心逻辑：区分长按与短按
脚本以 **450毫秒 (ms)** 为界限来判断你的操作意图：

* **短按**：按住时间小于 0.45 秒。
* **长按**：按住时间大于或等于 0.45 秒。

### 2. 具体行为规则

#### A. 当你【短按】CapsLock 时
系统会先判断当前的大写锁定状态（CapsLock Light）：

* **如果当前是大写状态 (Light On)：**
    * **行为**：关闭大写锁定，恢复为小写模式。
    * **目的**：快速退出大写模式。

* **如果当前是小写状态 (Light Off)：**
    * **行为**：发送 `Ctrl + Space` 组合键。
    * **目的**：通常用于切换输入法（例如在 Windows 默认设置或 macOS 风格设置中切换中/英输入法）。
